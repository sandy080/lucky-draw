# 🎁 AI-Powered Event Lottery Tool

An interactive, client-side lucky draw web application built efficiently through AI-assisted engineering via **Gemini Vibe Coding**.

## 🚀 Key Features
* **3 Custom Drawing Modes:** Supports Basic (Instant), Slot Machine (Animation), and Lucky Wheel (Canvas-based).
* **Procedural Sound Effects:** Uses native browser **Web Audio API** for real-time sound synthesis (100% copyright-free and zero external asset dependencies).
* **HR & Admin Friendly:** Supports smart range inputs (e.g., `1-100`, `A01-A50`) and instant Excel-compatible text export.
* **Zero Dependencies:** Pure HTML5, CSS3, and Vanilla JavaScript—fast, secure, and fully hosted on GitHub Pages.

## 🛠️ Tech Stack & Methodology
* **Frontend:** HTML5 (Canvas), CSS3, JavaScript (ES6)
* **Audio Engine:** Web Audio API (OscillatorNode, GainNode)
* **Development:** Developed via **Gemini Vibe Coding**, demonstrating rapid prototyping, AI-collaborative logic refinement, and efficient prompt-to-code iteration.
